# WSNM-WebService

## Installation
1) Install pip (pip3)
    - `sudo apt install python3-pip`
2) Install pipenv
    - `sudo -H pip3 install -U pipenv`
    - Install dependencies
        - `pipenv install` (from Pipfile)
3) Install Redis-Server
    - `sudo apt install redis-server`

## Configuration
- `config.py`
- `.env`

## Run
- `./run_flask`
- `./run_celery`
